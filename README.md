## `專案介紹`

幫助客人點餐與幫助店家出餐

#### `功能介紹`
⌞點餐介面:消費者可以將點餐資料送出<br>
⌞廚房介面：廚房可以接收消費者傳來的點餐資料<br>
⌞櫃台結帳介面：店家可以計算總金額，並移除舊資料<br>
